# Outcome for Python Lists & Tuples

## Create and display a list
**Expected Outcome:**
The program should create a list of numbers and display all the elements in the list.

**Actual Outcome:**
A list containing numbers is successfully created and printed on the screen in the same order as defined.

## Find the largest and smallest number in a list

**Expected Outcome:**
The program should identify and display the largest and smallest values present in the list.

**Actual Outcome:**
The program correctly finds the maximum and minimum values using built-in functions and displays them.

## Sort the list in ascending and descending order

**Expected Outcome:**
The list elements should be arranged in ascending order and descending order.

**Actual Outcome:**
The list is sorted successfully in both ascending and descending order.

## Convert list to tuple

**Expected Outcome:**
The list should be converted into a tuple and displayed.

**Actual Outcome:**
The list is successfully converted into a tuple, and the tuple is displayed.
