## Experiment
To create a basic web page using HTML that includes a heading, paragraph, image, hyperlink, ordered list, unordered list, and a table displaying student details.

**Expected Outcome**
After executing the HTML code, the web page should display a properly formatted heading and paragraph, an image, a clickable link, both ordered and unordered lists, and a table containing student information arranged in rows and columns.

**Actual Outcome**
The web page was successfully created and displayed in the browser. All elements such as the heading, paragraph, image, hyperlink, lists, and student details table appeared correctly and functioned as expected.
